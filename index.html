História da Web

(Voltar)

Tudo se inicia por volta de 1944 com o Mark I, que foi a primeira geração de computadores. Em seguida o Colossus, depois o Eninac, esse já um pouco mais automatico, ambos 1946. A segunda geração de 1959 a 1965, onde aconteceram as evoluções das válvulas para os transitores, o que ajudaram a diminuir o tamanho dos computadores. Com o circuito integrado (os chips), a terceira geração aconteceu de 1965 a 1970. Nesse circuito integrado tinha o transitor, capacitores e resitores. Com esses sistemas foram diminuindo ainda mais o tamanho dos computadores e aumentando a sua eficacia. A quarta geração foi a partir de 1971 até os dias de hoje, evoluindo cada vez mais.

Algumas pessoas dizem que estamos vivendo a quinta geração, que seriam os computadores quanticos, inteligencias artificiais, as criptomoedas e o metaverso. Mas tambem existem outras pessoas que dizem estar longe disso. O que são Clients?

(Voltar)

Clients é quando você consome a informação de algum lugar. Baseado na internet, o client é exatamente o dispositivo que está consumindo aquela informação na internet.

O que são Servers?

(Voltar)

Como o nome já diz, Servers é o servidor, ele serve informações aos clientes.Quando fazemos um site, precisamos fazer a hospedagem quando for publicar, significa que vai colocar o arquivo de internet no servidor.

Linguagens de Programação

(Voltar)

Linguagem de programação é uma linguagem escrita e formal que especifica um conjunto de instruções e regras usadas para gerar programas (software). Basicamente é uma ferramenta para resolver problemas. Segundo pesquisas da Redmonk, as 5 linguagens mais populares ate janeiro de 2022 são:

  1. Javascript
  2. Python
  3. Java
  4. PHP
  5. C#
Também tem a linguagem de programação que são executadas no servidor, alguns exemplos são PHP, JS, C#, Java, Ruby. Cada uma dessas linguagens tem a sua particularidade, umas mais fáceis, outras mais difíceis, mas no final praticamente todas tem o mesmo resultado. No lado cliente, a linguagem executada pelo navegador é o Javascript. O Javascript roda tanto no frontend quanto no backend.

Estutura básica do Html

(Voltar)

O Html consiste em 3 partes. A primeira é a tag html e dentro dela temos a tag head e body, que são a cabeça e o corpo do site. As maioria das tags possuem abertura e fechamento, são os sinais de maior e menor, e na de fechamento temos a barra indicando que fechou. Dentro do head temos a tag title, que é o título da página que vamos usar. O que fica dentro do head o usuário não vê, apenas o que está no body.

Listas ordenadas e não ordenadas

(Voltar)

Existem dois tipos de listas, são as ordenadas e as não ordenadas. As ordenadas são feitas com a tag ol, e dentro dela vem as filhas, no caso as tags li, que são os itens da lista. As listas não ordenadas são utilizadas as tags ul, nesse caso ao invés de números, a lista terá marcadores.
